template<operator_kind OpKind, typename Backend>
class arpack_solver¶

Main class template of ezARPACK’s API.

Instances of this class are used to solve all kinds of eigenproblems supported by ezARPACK.

tparam OpKind

Kind of eigenproblem to be solved.

tparam Backend

Tag type specifying what storage backend (matrix/vector algebra library) must be used by arpack_solver. The storage backend determines types of internally stored data arrays and input/output view objects exposed by methods of the class.
