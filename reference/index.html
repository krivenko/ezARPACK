List of C++ headers:

  • Specializations of the arpack_solver class template
    • ezarpack/solver_symmetric.hpp - symmetric real eigenproblems
    • ezarpack/solver_asymmetric.hpp - general real eigenproblems
    • ezarpack/solver_complex.hpp - general complex eigenproblems
  • Data storage backends
    • ezarpack/storages/base.hpp - definition of storage_traits structure
    • ezarpack/storages/eigen.hpp - Eigen 3
    • ezarpack/storages/blaze.hpp - Blaze
    • ezarpack/storages/armadillo.hpp - Armadillo
    • ezarpack/storages/ublas.hpp - Boost uBLAS
    • ezarpack/storages/triqs.hpp - TRIQS arrays
    • ezarpack/storages/xtensor.hpp - xtensor
  • Auxiliary headers
    • ezarpack/solver_base.hpp - primary arpack_solver class template
    • ezarpack/arpack_solver.hpp - include all arpack_solver specializations
    • ezarpack/arpack.hpp - Low level C++ interface to FORTRAN routines of ARPACK
    • ezarpack/version.hpp - ezARPACK version information
